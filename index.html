Projects

Selected technical projects imported from napath.com (blog excluded).

CheetahDB

2018–2020

GPU-first DBMS supporting SQL with client/server communication via TCP/IP and JSON. Heavy operators (join, group-by, search) are accelerated on GPUs.

Project source page ↗

Parallel Index Search for DBMS via GPUs

2018–2020

Designed dynamic GPU index structures, low-overhead memory allocation, and exact/range search optimizations with cache-aware grouping.

Project source page ↗

Support Measures for Frequent Patterns in Single-Graph Databases

2017–2020

Built high-performance support-measure computation mapped to optimization formulations, preserving counting quality while improving runtime.

Computing 2-Body Statistics on GPUs

2015–2018

Developed decomposition and tuning strategies, analytical models, and a framework to auto-generate high-performance GPU code for 2-BS tasks.

Tuple/Page Access Pattern Study in DBMS

2014

Modified PostgreSQL to track page and tuple access frequencies to support improved buffer management and reduced disk traffic.

GPU Keyword Search for Network Forensics

2012

Applied CUDA-based KMP skip-search for lawful interception/forensics workloads and investigated GPU parameter tuning for performance.
