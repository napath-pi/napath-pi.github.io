Algorithms for Computing 2-body Statistics on GPUs

2015–2018

This project designs and optimizes GPU algorithms for quadratic-cost two-body statistics. It includes decomposition methods, analytical parameter models, and an auto-generation framework for high-performance GPU implementations.
