Support Measures for Counting Frequent Patterns in Single-Graph Databases

2017–2020

Frequent pattern mining has been an attractive subject in data mining research for decades. Furthermore, as today, rapid growing network data is creating a great potential for knowledges discovery from graph data, especially single-graph databases as efficient structures representing these complex relationships are becoming more and more valuable and significant.

There are two problems in frequent pattern mining in a single-graph setting, support measure, and search scheme. In this project, we address the challenge of implement a high-performance polynomial and non-polynomial calculation of new support measures by mapping to the optimization problem. The outcome of the project is a software that can calculate a support measure in graph frequent pattern analysis with the linear time but still preserve the number of counting close to the actual value.
